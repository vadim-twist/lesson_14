FLEXBOX

Flexbox - це цілий модуль, що містить у собі набір властивостей та їх значень як для батька flex-контейнера, так і для його дочірніх елементів flex-елементів. Основною перевагою гнучкої FLEX розмітки є можливість змінювати ширину/висоту простір навколо та порядок flex-елементів, для того, щоб найкращим чином заповнити доступний простір flex-контейнера.

Flex

Дозволяє вказати параметри елемента, щоб він ефективно заповнював доступний простір. Елементи можуть бути розтягнуті пропорційно з урахуванням заданого співвідношення або стиснуті, щоб цілком вмістити всі елементи без переносів в один рядок.

flex-grow - 0 заборонено, 1 дозволено (за замовченням - 0). Визначає можливість елемента збільшуватись у розмірі, при необхідності заповнюючи весь flex-контейнер.
